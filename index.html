About

This application was created as a way to easily perform basic statistical analysis from data. You can either upload a file to the application or copy and paste the text from a tabular data file. This will be the data that is used to generate the statistics from. Additionally, the contents of the data file will be displayed in a table for the user to review. Below that, statistics will be displayed that summarize each column of the dataset.

When it is complete, the user will be able to drag and drop variables from the parsed dataset to select specific columns that they want in the table. Additionally, they will be able to specify the data-type of the variable and drag and drop the variables into different graphing widgets that will be able to generate visualizations of the data as well as widgets to run bivariate and multivariate statistical analyses.
